斐波那契数列通项三种解法（普通数列解法，线性代数，信号与系统）

May 6th, 2009

(hplonline)2009.5.6

因为公式很不好弄到网页上来。。所以完整的ＤＯＣ在这里下载。
下不了的同学可以直接找我要。

先给一个我觉得比较好看的结果：



这里只给出三种求得的结果。有点差异，其实是一样的。
按照相应的MATLAB程序给出。产生前10项做测试。

１。 高中数列方法

for n=1:10

p = (1-sqrt(5))/2 ;

q = (1+sqrt(5))/2 ;

x = q^(n-2)*(1-p)*(1-(p/q)^(n-1))/(1-p/q)+p^(n-1)

end


 ２。线性代数

for n=1:10

x=-(16/5)*sqrt(5)*(-(1/2+(1/2)*sqrt(5))^n+(1/2-(1/2)*sqrt(5))^n)/((1+sqrt(5))^2*(-1+sqrt(5))^2)

end


３。信号与系统

for n=1:10

q1 = (1-sqrt(5))/2 ;

q2 = (1+sqrt(5))/2 ;

c1 = -sqrt(5)/5 ;

c2 = sqrt(5)/5 ;

x = c1 * q1^n + c2 * q2^n

end


４。小结

这几个推导不一定是化到最简的式子，反正能转化成实际的编程语言使用就是了。

第1个推导所用基础知识最少，

但是是建立在能猜到数列最后造型的基础上得来的。

不属于比较通用的方法。

第2个用到的线性代数的知识也不深，

容易理解，推导严密。

从严密性来说，总觉得第3个欠妥。。

毕竟学信号的时候，搞得似是而非的。

但是从结果的表达来看，第3个最美观。

从运算上来说，这三个给出的通项都得使用浮点运算。

要得到准确的结果，可以套用2中的矩阵乘法的思想。

使用快速矩阵求幂，效率也很高。
