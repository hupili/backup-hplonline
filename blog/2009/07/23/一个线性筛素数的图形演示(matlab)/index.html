一个线性筛素数的图形演示(matlab)

Jul 23rd, 2009

(hplonline)2009.7.23

做完了之后，发现其实也并不直观到哪里去，
而且由于matlab绘图的原因，演示的范围很有限。
所以估计一下，知道的同学看了知道是啥意思，
不知道的同学也不用期望看这个就彻底理解了，
还是得去找找其他的详细解释来看看。

matlab实现的线性筛素数：

clear ;
N = 20 ;
marks = zeros(1,N) ;
primes = [] ;
for i = 2:N
    if ~marks(i)
        primes = [primes , i] ;
    end
    for j = 1:length(primes)
        if i * primes(j) > N
            break ;
        end
        marks(i * primes(j)) = 1 ;
        if mod(i , primes(j)) == 0
            break ;   
        end
    end
end
primes

在各个步骤中加入绘图演示的程序：

clear ;
T = 1 ;%单步间隔时间
N = 20 ;%演示到的点数
marks = zeros(1,N) ;
plot(1:N , ones(1,N) , ‘o’) ;
for i = 1:N
    text(i,0.9,int2str(i)) ;
end
axis([0 N+1 0 2]) ;
hold on ;
primes = [] ;
for i = 2:N
    quiver(i - 0.2 ,1.2 , 0.1 , -0.15 , 0) ;pause(T) ;
    if ~marks(i)
        primes = [primes , i] ;
        plot(i,1,’’) ;pause(T) ;
        text(length(primes) , 1.5 , int2str(i)) ;
    end
    plot(1:length(primes) , 1.4 , ‘og’) ;pause(T) ;
    for j = 1:length(primes)
        if i * primes(j) > N
            break ;
        end
        plot(j , 1.4 , ‘or’) ;pause(T) ;
        marks(i * primes(j)) = 1 ;
        plot(i * primes(j) , 1 , ‘xr’) ;
        plot(i * primes(j) , 1 , ‘or’) ;
        pause(T) ;
        if mod(i , primes(j)) == 0
            break ;   
        end
    end
end
primes
