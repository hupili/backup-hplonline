(hplonline)2009.8.4

知道大意后，详细内容还是要参考原文，

这篇的内容涉及到很多宏的定义，需要从头按顺序看，才能理解，要记住各个宏定义的时候的参数是什么意义。这篇在于给出一个功能上的完整集合，所以并非里面的宏是在具体使用时候的最优解决方案，很多时候，可以针对特殊情况进行改进的。但是通过这篇给出的宏的方式，比较容易构建程序，并且展开方便。

移动，复制，加法

最简单的程序，清零当前所指的格:

[-]

把当前格的数字移动到右边一格:

[ > + < - ]

通用的移动数字的框架为:

  • 把指针移动到源点
  • ”[” command
  • 把指针移动到目的点
  • ”+” command
  • （这一步可能对多个目的点相加）
  • 把指针移动到源点
  • ”-” command
  • ”]” command

定义 to(m) 为把指针移动到m

move(s,d) 表示移动s的数字到d, 可以写作:

move(s,d) = to(s) [ to(d) + to(s) - ]

定义 for(a) 和 next(a):

for(a) = to(a)[

next(a) = to(a)-]

move(s,d)

= for(s)

      to(d) +

    next(s)

把一个源点移动到两个目的点:

move2(s,d1,d2)

= for(s)

      to(d1) +

      to(d2) +

    next(s)-]

把s复制到t需要用到一个临时单元t:

copy(s,d,t) = move2(s,d,t) move(t,s)

以上的操作都默认了各个格子在操作之前是空的。

乘法和幂

把一个数变成两倍:

double(s,d) = move2(s,d,d)

用 constant(n) 表示在当前格设置常数n

那么s乘以常数n可以写作:

multiplyconst(s,d,n)

= for(s)

      to(d) constant(n)

    next(s)

两个数相乘：

times(s1,s2,d,t)

= for(s1)

      copy(s2,d,t)

    next(s1)

    zero(s2)

注意，在以上以及以后的定义中，都认为一个操作完成后源格被置零。

乘幂，就是重复乘法就行了:

power(x,p,d,t1,t2,t3)

= to(d) +

    for(p)

        copy(x,t1,t2)

        times(d,t1,t2,t3)

        move(t2,d)

    next(p)

    zero(x)

（红色部分为hplonline更正，因为copy宏的定义要用到一个临时格）

If-then-else分支结构

之前实现了循环，[]可以进行判断，而分支可以视作只执行一次的循环，所以按照循环的结构，在分支内部把控制循环真值的格清零即可。定义如下符号:

zero(a) = to(a) [-]

if(a) = to(a) [

endif(a) = zero(a) ]

实现else结构，只需增加一个临时变量，首先置1，进入if分支的时候清零，则在下一次判断不进入else分枝:

ifelse(a,t) = inc(t) if(a) dec(t)

else(a,t) = endif(a) if(t)

endelse(t) = endif(t)

最后一个宏也可写作:

endelse(t) = dec(t) ]
