遍历目录的模块

Sep 9th, 2009

(hplonline)2009.9.9

经常会用到的操作，
遍历某个文件夹（包括其子目录），
然后对其中的文件进行某种操作。

SearchDir函数可以完成这个功能，
传入OPFUNC类型的回调函数即可。
OPFUNC可以得到每个文件名，并决定作何操作。
下面的程序演示了输出d:\下的所有文件名的方法。
关于路径的习惯，根目录后面是带’'的，
而其他目录是不带’'的。
我也觉得这是个很别扭的事情，
但是前人都这么搞的，
也就跟着这么搞。

有的时候，最初的设计不周，
导致后人多写许多句if。。。

—————–

#include <windows.h>
#include <stdio.h>
#include <string.h>

char Dir[] = “d:\” ;

typedef void (*OPFUNC)(char *filename) ;

void myOperation(char *filename){
    puts(filename) ;
}

void SearchDir(char *dir , OPFUNC func){
    WIN32_FIND_DATA wfd ;
    char szCurDir[MAX_PATH] ;
    char szFind[MAX_PATH] ;
    strcpy(szCurDir , dir) ;
    int len = strlen(szCurDir) ;
    //is root directory or not
    if ( szCurDir[len - 1] != ‘\’ ) {
        strcat(szCurDir , “\”) ;   
    }
    strcpy(szFind , szCurDir) ;
    strcat(szFind , “*.*”) ;

    HANDLE hFind=FindFirstFile(szFind , &wfd) ;
    if ( hFind == INVALID_HANDLE_VALUE ) {
        return ;
    }else{
        do{
            //filtering the current directory and its parent
            if ( wfd.cFileName[0] == ‘.’ ) continue ;
            char szFileName[MAX_PATH] ;
            sprintf(szFileName , “%s%s” , szCurDir , wfd.cFileName ) ;
            //this is a directory and do recursion
            if ( wfd.dwFileAttributes & FILE_ATTRIBUTE_DIRECTORY ) {
                SearchDir(szFileName , func) ;
            }else{
                (*func)(szFileName) ;
            }
        }while ( FindNextFile(hFind , &wfd) ) ;
    }
    FindClose(hFind) ;
}

int main(){
    char sDir[MAX_PATH] ;
    SearchDir(Dir , myOperation) ;
    system(“pause”) ;
    return 0 ;
}
