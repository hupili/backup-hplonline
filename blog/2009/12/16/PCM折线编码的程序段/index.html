PCM折线编码的程序段

Dec 16th, 2009

(hplonline)2009.12.16

最近赶通原作业，顺便做了。
其实上学期的信息论的时候就想写的，
又是一直放在那没动。。
这回要整作业，写出来好用。。
mu律的程序类似。

function main
clc ;
clear ;
pcm_a(0.12)
pcm_a(0.3)
pcm_a(-0.7)
return

function code = pcm_a(x)

if x >= 0
    sign = ‘1’ ;
else
    sign = ‘0’ ;
    x = -x ;
end

b(1) = 1 / 128 ;
b(2) = 1 / 128 ;
c(1) = 0 ;
c(2) = b(1) ;
for i = 3:8
   b(i) = b(i - 1) * 2 ;
   c(i) = c(i - 1) + b(i - 1) ;
end

seg = sum(c <= x) ;
% [b’ , c’]’
% x
% c(seg)
% b(seg)
offset = 16 * (x - c(seg)) / b(seg) ;

code = [sign , dec2bin(seg - 1, 3) , dec2bin(fix(offset) , 4)] ;

return
