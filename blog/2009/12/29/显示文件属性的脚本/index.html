显示文件属性的脚本

Dec 29th, 2009

(hplonline)2009.12.29

三个星期前做微机原理实验的时候，
问某牛应该怎么查看文件属性。
感觉相当惭愧啊。。
都大三的人了，连linux都不会用。

最近偷点闲，看了下鸟哥的那本书。
还不错。。知道点基本的东西了。
就写这么个查看文件属性的脚本来练练手。

主要是if,for,array，ls,cut的用法

另外对vi还是很手生，用起来特慢。
估计得啥时候刷几场比赛来刺激一下熟练程度。
不过在那之前貌似还要学下makefile和gdb，
就可以甩掉IDE在linux下工作了。。

脚本“/usr/bin/attr”：

#!/bin/bash
#created by hpl
#2009/12/28
#test the attributes of a file

if [ “$1” = “” ] ; then
    echo “usage: attr <filename>”
    exit 1
fi
if [ ! -e $1 ] ; then
    echo “file "$1" not exist!!”
    exit 2
fi

echo “analyzing file:"$1"”

if [ -d $1 ] ; then
    echo “it’s a directory”
    exit 0
fi

declare -a usr=(owner group other)
declare -a rt=(read write execute)

for (( i=0 ; i<=2 ; i=i+1 ))
do
    echo “${usr[$i]}:”
    for (( j=0 ; j<=2 ; j=j+1))
    do
        declare -i k=i*3+j+2
        ex=`ls -l $1 | cut -c $k`
        if [ “$ex” = “-” ] ; then
            echo “    can’t ${rt[$j]}”
        else
            echo “    can ${rt[$j]}”
        fi
    done
done
