环境变量函数（C,Linux）

Jan 8th, 2010

(hplonline)2010.1.8

linux下c的stdlib里面带有几个操作环境变量的函数。
getenv , putenv , setenv , unsetenv
其中putenv的功能在setenv下面显得冗余了。
不过调用方式有点小区别，
用不着怎么描述，看代码就知道了。

另外想起了弄个宏
#define S(x) printf(“%s\n” , #x) ; x ;
这样在shell上观察语句效果比较方便，
便于学习各种函数时使用。

示例代码：

#include <stdio.h>
#include <stdlib.h>

#define S(x) printf(“%s\n” , #x) ; x ;

void print_env(char *name){
    char *buf ;
    buf = getenv(name) ;
    printf(“%s:\n%s\n\n” , name , buf) ;
}

int main(){
    char *buf ;
    //test of getenv in function print_env
    //show the variable PATH
    S(print_env(“PATH”) ;)
    //print a variable that not exists
    print_env(“HPL”) ;
    //try to put an variable through putenv
    S(putenv(“HPL=1”) ;)
    print_env(“HPL”) ;
    //try to put an variable through setenv
    S(setenv(“HPL2” , “2” , 1) ;)
    print_env(“HPL2”) ;
    //try the difference of override flag of setenv
    S(setenv(“HPL2” , “3” , 0) ;)
    print_env(“HPL2”) ;
    S(setenv(“HPL2” , “3” , 1) ;)
    print_env(“HPL2”) ;
    //try removing variables
    S(unsetenv(“HPL”) ;)
    S(unsetenv(“HPL2”) ;)
    print_env(“HPL”) ;
    print_env(“HPL2”) ;

    S(printf(“aaaa”)) ;

    return 0 ;
}

看通原很郁闷，
正好vim配方便了，手痒。
