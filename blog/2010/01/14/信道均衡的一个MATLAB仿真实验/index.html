信道均衡的一个MATLAB仿真实验

Jan 14th, 2010

(hplonline)2010.1.14

这学期学了通原，
虽然知道点基本概念，
不过大多会在数日之后搞忘。
想起前天居然没考信道均衡，
相当不解。。
毕竟信道均衡是我们这次新教材添加进的内容。

再想下平时上课看的课件也有很多老教材的影子，
可见整个事情就是新瓶旧酒。
一样的老师，一样的课件，一样的实验，一样的考试，
就换了本书。

好不容易所有试都考完了，
蛋疼一下，做个信道均衡的实验。
用的就是最简单的迫零均衡，
不过效果似乎没有想象中的那么好，
均衡前后听着都是嗡嗡的。
主要在通过2N+1抽头的均衡器后，
整个系统响应中间的2N+1个系数满足要求了，
但之外的系数反而会由零变成有值的，
同样引入ISI。

示例程序和音乐下载。

程序：

（请不要使用太长的音乐，否则会filter很久）

function main
[y fs nbits] = wavread(‘origin.wav’) ;

% wavplay(y , fs) ;

N = 100 ;
h = rand(1 , 2 * N + 1) ;
yy = filter(h , [1] , y) ;
wavplay(yy , fs) ;

c = get_equiv(h) ;
yyy = filter(c , [1] , yy) ;
wavplay(yyy , fs) ;

return ;

%forcing to zero algorithm
function c = get_equiv(h)
M = length(h) ;
N = (length(h) - 1) / 2 ;
h = fliplr(h) ;
H = [] ;
for ii = 1:N
tmp = h(N - ii + 2 : M) ;
tmp = [tmp , zeros(1 , N + 1 - ii)] ;
H = [H ; tmp] ;
end
H = [H ; h] ;
for ii = 1:N
tmp = zeros(1 , ii) ;
tmp = [tmp , h(1:M-ii)] ;
H = [H ; tmp] ;
end   
%hE = H * c
hE = [zeros(N , 1) ; 1 ; zeros(N , 1)] ;
c = H \ hE ;
return ;
