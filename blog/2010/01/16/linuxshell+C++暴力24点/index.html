Linux shell+C++暴力24点

Jan 16th, 2010

(hplonline)2010.1.16

24点的程序可以说很多人都写过，
用不同的语言，不同的思路。
即使都是全枚举，很多细节也会不一样。

写了一天的文档，累得不行了。。
以前没写过，正好最近搞linux，就用SHELL和C++来暴力一个24点。

思路：

linux的shell有个好处就是可以直接按整形算符计算表达式。
于是可以把表达式构造好，然后交给shell就行了。
虽然直接用shell也可以构造表达式，
但毕竟对C熟悉一点，觉得shell的语法很别扭，
故就用C++来产生表达式。

我用的方法会产生出重复的，
但实际上是交换律、结合律、括号法则等意义下的重复，
比如，3+4，4+3在我这里就认为是两个表达式。
要达到绝对化的不重复，写起来就很烦了。
再说，所有合法的表达式都是在四则运算意义下的重复。
24=(4-2)*(3+9)
24=((2*4)*9)/3
就像上面两个式子整体看上去都是一个意思，怎么能说没重复呢？
如果我用手把他们的细节遮住，你只可能看到两个24。

首先，对四个数进行全排列。
1 2 3 4
1 2 4 3
….
4 3 2 1
这个用next_permutation就出来了。

然后归纳模式，我觉得应该只有下面这几种吧：
    “((%d%c%d)%c%d)%c%d” ,
    “(%d%c%d)%c(%d%c%d)” ,
    “(%d%c(%d%c%d))%c%d” ,
    “%d%c((%d%c%d)%c%d)” ,
    “%d%c(%d%c(%d%c%d))” ,
在四个%d的地方，依次填上四个数。
在有%c的地方填上运算符。
只考虑了+-*/四种，就依次带入就行了。

够暴力吧，哈哈。

用来计算表达式的shell脚本：cal24.sh

生成表达式的C++程序：


#include <iostream>
#include <algorithm>
#include <stdio.h>

using namespace std ;

char *patterns[] = {
    “((%d%c%d)%c%d)%c%d” ,
    “(%d%c%d)%c(%d%c%d)” ,
    “(%d%c(%d%c%d))%c%d” ,
    “%d%c((%d%c%d)%c%d)” ,
    “%d%c(%d%c(%d%c%d))” ,
} ;
const int P_NUM = 5 ;

void brute_force(int *a){
    const char c[] = “+-*/” ;
    char i[3] , j ;
    char buffer[100] , cmd[150] ;
   
    for ( i[0] = 0 ; i[0] < 4 ; i[0] ++ ){
        for ( i[1] = 0 ; i[1] < 4 ; i[1] ++){
            for ( i[2] = 0 ; i[2] < 4 ; i[2] ++){
                for ( j = 0 ; j < P_NUM ; j ++ ) {
                    sprintf(buffer , patterns[j] , a[0] , c[i[0]] ,
                        a[1] , c[i[1]] , a[2] , c[i[2]] , a[3] , c[i[3]]) ;
                    sprintf(cmd , “./cal24.sh "%s" 2> /dev/null” , buffer) ;
                    //printf(cmd) ;
                    system(cmd) ;
                }              
            }
        }
    }
}

int main(){
    int a[4] ;
    int i ;
    printf(“please input 4 numbers , seperate by blanks:\n”) ;
    for ( i = 0 ; i < 4 ; i ++ ){
        scanf(“%d” , &a[i]) ;
    }
    sort(a , a + 4) ;
    do{
        brute_force(a) ;      
        /*    for ( i = 0 ; i < 4 ; i ++ ){
        printf(“%d ” , a[i]) ;
        }
        printf(“\n”) ;
        */
    }while ( next_permutation(a , a+4)) ;
    return 0 ;
}
