<p>function main<br/>
clc ;<br/>
clear ;<br/>
pcm_a(0.12)<br/>
pcm_a(0.3)<br/>
pcm_a(-0.7)<br/>
return</p>
<p>function code = pcm_a(x)</p>
<p>if x &gt;= 0 <br/>
     sign = '1' ;<br/>
else <br/>
     sign = '0' ;<br/>
     x = -x ;<br/>
end</p>
<p>b(1) = 1 / 128 ;<br/>
b(2) = 1 / 128 ;<br/>
c(1) = 0 ;<br/>
c(2) = b(1) ;<br/>
for i = 3:8<br/>
    b(i) = b(i - 1) * 2 ;<br/>
    c(i) = c(i - 1) + b(i - 1) ;<br/>
end</p>
<p>seg = sum(c &lt;= x) ;<br/>
% [b' , c']'<br/>
% x<br/>
% c(seg)<br/>
% b(seg)<br/>
offset = 16 * (x - c(seg)) / b(seg) ;</p>
<p>code = [sign , dec2bin(seg - 1, 3) , dec2bin(fix(offset) , 4)] ;</p>
<p>return</p>
